Logging strategy, what would we like to log?

We have four categories:
- Error: on Error on the highest level, we log these.
- Warning
- Information
- Debug

Ideas for logging:
- When calling all routing functions, we could create a Debug log or Information log about how much time any specific step took:
    - GenerateHeadland took "x"ms
    - GenerateMainland took "x"ms
- This could be either Information or Debug.



Warning ideas:
- Any warning codes that we will return should also be logged as `Warning`.
  Ideally, we log them directly whenever the warning happens in the AgxRouteLib.
